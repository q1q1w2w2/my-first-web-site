HTML

Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications.

HTML stands for HyperText Markup Language. It is a standard markup language for web page creation.

It allows the creation and structure of sections, paragraphs, and links using HTML elements

(the building blocks of a web page) such as tags and attributes.
